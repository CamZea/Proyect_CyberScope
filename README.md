# CyberScope 🛡️
Plataforma de reconocimiento y evaluación de seguridad desarrollada en Python.

## ¿Qué hace?
Con un solo comando escanea una red, calcula el riesgo de cada servicio
expuesto y genera un reporte PDF profesional automáticamente.

## Módulos completados
- Network Scan — escaneo real con Nmap
- Risk Scoring — análisis de riesgo por servicio y versión
- Report Generator — reporte PDF automático
- OSINT — en desarrollo
- Threat Intelligence — en desarrollo

## Tecnologías
Python · Nmap · Docker · ReportLab

## Laboratorio
3 contenedores Docker con servicios vulnerables reales:
- DVWA (Apache 2.4.25)
- FTP Server (vsftpd 2.0.8)
- SSH Server (OpenSSH)

## Uso
```bash
py main.py
```
